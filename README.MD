
##**README **

##**Run tests with Gradle:**

```
./gradlew clean test
```

##**Run tests with Gradle and specific tags:**

```
./gradlew clean test -DincludeTags='smoke'
```

##**Run tests with Maven:**

```
mvn clean verify
```
##**Run tests with Maven and specific tags:**

```
mvn clean verify -DincludeTags='smoke'
```
